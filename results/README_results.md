# Results

This folder contains output tables generated from the modeling and risk-extension notebooks.

- `model_performance_ba_gp.csv`: Cross-validated performance of full-feature BA_GP prediction models.
- `model_comparison_baseline_vs_full.csv`: Comparison between total-Cd-only baseline models and full-feature models.
- `feature_importance_ba_gp.csv`: Permutation importance results for the selected Random Forest model.
- `ba_gp_summary_by_method.csv`: Summary statistics of BA_GP by in vitro digestion method.
- `ba_gp_summary_by_type.csv`: Summary statistics of BA_GP by soil source type.
- `h2_method_type_kruskal_results.csv`: Kruskal-Wallis test results for Method and Type effects.
- `bioaccessible_cd_risk_data.csv`: Sample-level observed and predicted bioaccessible Cd indicators.
- `bioaccessible_cd_risk_by_type.csv`: Summary of predicted bioaccessible Cd risk groups by soil source type.
